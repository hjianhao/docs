# FAQ

## 如何指定WSL实例的目录

[参见LxRunOffline](#LxRunOffline)

``` bash
LxRunOffline install -n <WSL名称> -d <安装系统的路径> -f <镜像文件路径>\xxx.tar.gz -s
```

其中：

-n是安装的系统名称，可自定义；

-d 是安装系统的目录；

-f是之前下载的镜像文件路径；

另外加入-s参数可在桌面创建快捷方式，可以不加。
